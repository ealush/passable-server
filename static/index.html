Passable on the server

Hey, thanks for checking out Passable. This is an example server that demonstrates Passable's basic capabilities.

It doesn't do much other than ansewr POST requests to the following routes:

  • /example/needful_things
  • /example/curse_purge_plus

A passing request for 'needful_things' could be: 

{
    "testData": {
        "MrBeauregard": {
            "says": "Marmalade is served"
        },
        "Snuffles": {
            "asks": "Where are my testicles, Summer?"
        },
        "Pilot": {
            "trigger": "none"
        }
    }
}

A passing request for 'curse_purge_plus' could be: 

{
    "testData": {
        "GetSchwifty": {
            "say": "Show me what you got"
        },
        "RickSanchez": {
            "shout": "Wubba lubba dub-dub!"
        },
        "BlipsAndChitz": {
            "game": "Roy1"
        }
    }
}

To see the code running on this server (not-interesting), head over to Passable-Server.
